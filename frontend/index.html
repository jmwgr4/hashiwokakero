🌉 Hashiwokakero - Bridge Builder

🎯 Goal: Connect all islands into a single connected group. Each number (1-8) shows how many bridges that island needs.

🎮 Controls: Click and drag from one island to another to create bridges. Bridges must be orthogonal and cannot cross.

💡 Hints: Use the "Hint" button for assistance, but note that it adds a 15-second penalty to your time!
